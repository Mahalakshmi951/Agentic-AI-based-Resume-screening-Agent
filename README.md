# 🤖 Agentic AI-Based Resume Screening System

## 📌 Project Overview

This project is an AI-powered Resume Screening System that automatically analyzes resumes and shortlists candidates based on job requirements.

It uses Agentic AI techniques to extract information, evaluate skills, and rank candidates efficiently.

👉 [Watch Demo Video](./demo%20video%20ai%20resume%20screening.mp4)

## 🚀 Features

* 📄 Resume Parsing (PDF/Doc)
* 🧠 AI-based Skill Extraction
* 🎯 Candidate Ranking System
* 📊 Automated Screening
* 📧 (Optional) Email Response System

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Mahalakshmi951/Agentic-AI-based-Resume-screening-Agent.git
```

### 2️⃣ Go to project folder

```
cd Agentic-AI-based-Resume-screening-Agent
```

### 3️⃣ Install dependencies

```
pnpm install
```

### 4️⃣ Run the project

```
pnpm dev
```

---

## 🔐 Environment Variables

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key_here
```

---

## 📸 Output
The system:
- Reads resumes
- Extracts skills like Python, Java, SQL
- Matches with job description
- Classifies candidates into:
  - ✅ Selected
  - ❌ Rejected
  - ⚠️ Maybe (Needs further review)
- Exports all results into an Excel file
